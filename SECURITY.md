# Security Policy

## Supported Versions

Below are the versions of **ChessGame-Elixir** currently supported with security updates:

| Version | Supported          |
| -------- | ------------------ |
| 1.x.x    | :white_check_mark: |
| 0.x.x    | :x:                |

Security patches are provided only for the latest major release (1.x.x).  
Older versions may still work but will not receive official fixes.
